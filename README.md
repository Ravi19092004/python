# Python Practice Scripts

This repository contains Python scripts for practice and utility.

## Files

### `goodwish.py`
A simple script that displays the current hour and a corresponding greeting.

**Output:**
1. Prints the current hour (0-23).
2. Prints a message based on the time:
   - **Morning** (00:00 - 11:59): "Good Morning!"
   - **Afternoon** (12:00 - 17:59): "Good Afternoon!"
   - **Night** (18:00 onwards): "good night!"

## Usage

To run the script, ensure you have Python installed and run the following command in your terminal:

```bash
python "goodwish.py"
```
